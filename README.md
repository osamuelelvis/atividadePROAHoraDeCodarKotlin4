## 🚀 Como executar

1. Clone este repositório:
```bash
git clone https://github.com/osamuelelvis/atividadePROAHoraDeCodarKotlin4.git
```
2. Abra em uma IDE (ex: IntelliJ IDEA)
3. Execute os arquivos `.kt`
